# cat_vasya_game
Ролевая игра от кота Василия: пошаговая боевка, классы персонажей и система заклинаний на Python
О проекте

Cat Vasya Game—учебный проект по принципам объектно-ориентированного программирования на Python. Реализует архитектуру классов для пошаговой ролевой игры с элементами:

Базовый абстрактный класс `Unit` с шестью характеристиками
Класс `Character` с поддержкой трёх классов: воин, маг, охотник
Система заклинаний `Spell` с расходом маны
Уникальные формулы расчёта урона, защиты и здоровья для каждого класса
Полное соответствие стилю PEP-8

Цель проекта

Демонстрация навыков работы с:
Абстрактными классами и наследованием
Инкапсуляцией логики внутри классов
Обработкой исключений и валидацией данных
Аннотациями типов и документированием кода

Технологии

Python 3.10+
Модуль `abc` для абстрактных классов
Type hints для статической типизации
Структура проекта по принципам модульности
